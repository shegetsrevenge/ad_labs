Лабораторні роботи з аналізу даних
Цей репозиторій містить лабораторні роботи з аналізу даних, виконані в рамках курсу. Кожна лабораторна робота розташована в окремій директорії.

Структура репозиторію
lab2/: Друга лабораторна робота, яка містить скрипт lab2(2).ipybn для обробки даних.
lab3/: Третя лабораторна робота, яка включає інтерактивний веб-додаток на основі Streamlit (lab3.py) для аналізу даних із файлу full.csv.
Встановлення та використання
Встановіть необхідні бібліотеки:
Копіювати
pip install streamlit pandas plotly
Для запуску третьої лабораторної роботи:
Копіювати
cd lab3
streamlit run lab3.py
Переконайтеся, що файл full.csv розташовано в директорії lab3/.
Вимоги до даних
Файл full.csv має містити стовпці: Year, Week, SMN, SMT, VCI, TCI, VHI, PROVINCE_ID.
